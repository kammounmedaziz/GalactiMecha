# GalactiMecha Frontend

Frontend scaffold for GalactiMecha (React with Express.js).

To run locally:

1. cd web
2. npm install
3. npm run build  # Build the React app
4. npm start      # Start the Express server to serve the build

For development (React dev server):

- npm run dev

This is a minimal React frontend with an Express.js server for the GalactiMecha project. No real features yet.
